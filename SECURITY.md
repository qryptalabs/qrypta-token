# Security Policy

## Scope

This repository contains the public smart contract source code, ABI, and technical configuration for the Qrypta token deployment.

## Current Security Posture

- Public contract source code is available in this repository.
- Main contract deployments are publicly verifiable on BscScan and Etherscan.
- The implementation uses OpenZeppelin-based components.
- A formal external audit report is not yet published in this repository.

## Responsible Disclosure

If you identify a security issue, please do not disclose it publicly first.

Please report security findings to:

**contact@qryptalabs.com**

Include:
- affected contract or file
- impact summary
- reproduction steps
- suggested remediation if available

## Notes

Security hardening, external review, and ecosystem-facing security processes remain active priorities for the project.
